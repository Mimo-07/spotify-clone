## Run this app

This web application is based on [Spotify API](https://developer.spotify.com/documentation/web-api), authorization is based on **Authorization code with PKCE extension** [reference](https://developer.spotify.com/documentation/web-api/tutorials/code-pkce-flow)

To develop application hosting url must be same as the redirect-uri mentioned in the reference.
